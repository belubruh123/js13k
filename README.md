# JS13K Game Project

This repository contains my entry for the [JS13KGames](https://js13kgames.com) competition.
It builds on top of the
[js13k-typescript-starter-webgl](https://github.com/roblouie/js13k-typescript-starter-webgl)
provided by [roblouie](https://github.com/roblouie). The starter kit supplies the build
pipeline and WebGL setup so I can focus on creating the game.

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server with hot module reloading:

   ```bash
   npm run serve
   ```

3. Create the production build (outputs a zipped file under `dist/`):

   ```bash
   npm run build
   ```
